## طريقة التشغيل

ستحتاج الى مكتبات بايثون لتشغيل البرنامج

المكتبات المتطلبة:

- PyQt5 للواجهة
- Requests لاستقبال خدمات API
- Geocoder لمعرفة موقعك بإستخدام IP المحلي الخاص بك

لتحميلهم:

- Windows:

```bash
python -m venv .venv
.venv\Scripts\activate.bat

pip install -r requirements.txt
```

- Mac & Linux:

```bash
python3 -m venv .venv
source .venv/bin/activate

# if you're using the fish shell
source .venv/bin/activate.fish

pip install -r requirements.txt
```

بعدها قم بتشغيل الملف بشكل عادي

يرجى تحميل الخط الموجود بنفس مجلد ملف البايثون لضمان التجربة الكاملة

## طريقة التثبيت

للتثبيت سوف تحتاج الى `pip` لتثبيتها عبر pypi

قم بكتابة

```bash
pip install sulatk==1.0.0
```

وسوف يثبت معك جميع الحزم المتطلبة لتشغيل الودجت
